API Gateway

Welcome to the University Room Booking System API Gateway. This gateway routes requests to the appropriate microservices.

Available Services

  • User Service - Handles user authentication and management
  • Booking Service - Manages room bookings and availability

API Endpoints

  • /api/auth/** - Authentication endpoints
  • /api/admin/** - Admin management endpoints
  • /api/bookings/** - Booking management endpoints
  • /api/rooms/** - Room management endpoints

For the frontend application, please visit the home page.
